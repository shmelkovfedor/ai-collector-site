# 📝 Как адаптировать сайт под себя

Этот файл — для партнёра/коллеги, который хочет изменить **тексты, цены, контакты** на сайте, сохранив дизайн (шрифт Geist, цвета navy + coral, анимации, voice-wave, кнопку «Запустить разговор»).

Никакого билда, фреймворков, npm — **один HTML файл, открывается в браузере**.

Скачай ZIP одним кликом:

👉 **https://github.com/shmelkovfedor/ai-collector-site/archive/refs/heads/main.zip**

## 2. Открыть в редакторе

Открой `index.html` в любом редакторе:
- **VS Code** (бесплатный): https://code.visualstudio.com/
- **Sublime Text**
- В крайнем случае — Блокнот / TextEdit

Открой файл параллельно в **Chrome** (просто двойной клик на `index.html`) — увидишь сайт локально, без интернета. После каждой правки нажми **Cmd+R** (Mac) / **F5** (Windows) — увидишь обновление.

## 3. Что менять — что НЕ менять

### ✅ Что можно безопасно менять

Всё **между тегами** `>текст<`. Открой `index.html`, найди строку **~870** (примерно середина файла) — там начинается `<body>`. Дальше идёт весь контент.

Примеры:

```html
<!-- ДО -->
<h1 class="hero-headline">AI-агент для коллекторских <span class="accent">агентств</span></h1>

<!-- ПОСЛЕ -->
<h1 class="hero-headline">AI-агент для вашей <span class="accent">компании</span></h1>
```

### ⛔ Что НЕ трогать

| Не менять | Где | Почему |
|---|---|---|
| Блок `<style>...</style>` | Строки 35–860 | Это все цвета/шрифты/анимации |
| Блок `<script>...</script>` | Строки 1512–1640 | Это reactive voice-wave + scroll-reveal |
| `class="..."` и `id="..."` | Везде | Привязка к стилям, поломается дизайн |
| Структура тегов (`<div>`, `<section>`) | Везде | Сетка сайта |

**Правило простое:** меняй только текст между `>` и `<`. Не трогай атрибуты `class=`, `id=`, `href=` если не уверен.

## 5. Проверить локально

После каждой группы правок:

1. **Cmd+S** — сохрани файл
2. Перейди в Chrome (где открыт `index.html`)
3. **Cmd+R / F5** — обнови страницу
4. Проверь что:
   - Текст обновился
   - Дизайн не сломан (цвета на месте, анимация работает)
   - Скрол снизу вверх работает плавно
   - Нажми «Запустить разговор» — волна голоса в hero должна «оживать»

Если что-то сломалось — **Ctrl+Z** до состояния когда работало.

## 7. Как развернуть свою версию (опционально)

Если хочешь свой домен и хостинг:

**GitHub Pages (бесплатно, ~5 минут):**
1. Создай GitHub аккаунт: https://github.com/signup
2. Создай новый репо (название = `your-name-site`)
3. Загрузь туда `index.html` + `assets/`
4. Settings → Pages → Source = `main` / root
5. Через 2 мин сайт будет на `https://your-name.github.io/your-name-site/`

**Если нужен свой домен** (например, `vashacompany.ru`):
- Купи домен на reg.ru / nic.ru (~600₽/год)
- В DNS укажи CNAME на `your-name.github.io`
- В GitHub Pages settings добавь Custom Domain
